# Kayuputi Bilingual Menu Cart

Static bilingual Traditional Chinese / English menu renderer with a simple cart calculator.

- Data source: `menu.bilingual.json`
- Renderer: `index.html` + `app.js` + `styles.css`
- Prices are in IDR and include 21% service charge and government tax.

## Local preview

```bash
python3 -m http.server 8765
open http://localhost:8765/
```
